Paso 0 — Antes de empezar

¿Qué app usar?

NXP

NFC TagWriter by NXP RECOMENDADA

Android · Gratis · NXP Semiconductors

La app oficial del fabricante del chip. Rápida, confiable y gratuita. Permite escribir, leer y bloquear tags. Es la que usamos en producción QUIE.

NFC

NFC Tools Pro

Android + iOS · ~$4 USD

La más completa del mercado. Permite escribir, leer, bloquear y ver todos los datos del chip. Ideal si trabajas también desde iPhone.

iOS

NFC.cool Tools

iOS iPhone 7+ (iOS 13+) · Gratis

Opción gratuita para iPhone. Puede leer y escribir URIs en NTAG213. Menos funciones que NFC Tools Pro pero sirve para producción básica.

📌 Importante: El NFC debe estar ACTIVADO en tu teléfono.
Android: Ajustes → Conexiones → NFC → Activar
iPhone: Funciona automáticamente en iPhone 7 o superior con iOS 11+

Proceso completo

Programar un Tag — Paso a Paso

1

Preparar el CSV del lote

Descarga el CSV desde el panel admin. Cada fila es un código con su URL.

  • Ve al Panel Admin → Lotes → CSV
  • Abre el archivo en Excel o Google Sheets
  • Ordena los tags físicos igual que el CSV
  • Trabaja de izquierda a derecha, fila por fila
💡 El CSV tiene: codigo_nfc, url_landing, modelo, color. La columna que vas a copiar es url_landing.
2

Abrir NFC TagWriter by NXP

Crear un nuevo dataset de tipo URI con la URL del primer código.

  • Abrir la app → "Write tags"
  • Tocar "New dataset"
  • Seleccionar "URI"
  • Pegar la URL del CSV:
https://quie-nfc-platform.vercel.app/v/QUIE-A7B3X9-42
  • Tocar "Save & Write"
💡 Copia la URL completa incluyendo el https://
3

Acercar el tag al teléfono

Mantener el chip plano y quieto bajo el teléfono hasta escuchar confirmación.

  • Coloca el tag NFC sobre una superficie plana
  • Acerca el centro o parte trasera superior del teléfono
  • Mantén quieto 1-2 segundos
  • La app mostrará: "Tag written successfully"
Atención: no mover el teléfono mientras escribe. Si falla, repetir. Después de 3 fallos, el tag puede estar dañado.
4

Verificar la escritura

Leer el tag escrito y confirmar que la URL es correcta antes de bloquear.

  • Cerrar la pantalla de escritura
  • Acercar el teléfono al mismo tag
  • Debe aparecer notificación del navegador
  • Confirmar que la URL coincide con el CSV
💡 También puedes abrir NFC Tools, leer el tag y ver la URL en la pestaña "Records".
5

Bloquear el tag (anti-falsificación)

Una vez verificado, bloquear para que nadie pueda sobreescribirlo.

  • En NFC TagWriter: "Other functions" → "Lock tag" → Confirmar
  • En NFC Tools Pro: Leer tag → "More" → "Make read-only"
Atención: el bloqueo es IRREVERSIBLE. Un tag bloqueado con URL incorrecta debe descartarse. Verificar ANTES de bloquear.
6

Pegar el tag en la pieza

Posición interior, zona de zona interior discreta, plano y sin interferencias metálicas.

  • Interior de la pieza, zona interior discreta frontal
  • Evitar costuras metálicas — interfieren con la señal
  • Si el tag no tiene adhesivo: usar cinta doble faz industrial
  • El tag debe quedar completamente plano
  • Hacer escaneo final desde el exterior para confirmar lectura
💡 Alternativa: bajo la etiqueta de talla/marca interior.
7

Marcar como listo y continuar

Avanzar al siguiente código del CSV y repetir desde el paso 2.

  • Marcar la fila del CSV como OK completada
  • Pasar al siguiente código en el CSV
  • Repetir desde el paso 2
  • Cada tag toma ~15 segundos
💡 Tiempo estimado: 50 tags = 12-15 min · 100 tags = 25-30 min

Resolución de problemas

Errores Comunes

Tag no detectado

Causa: NFC desactivado en el teléfono o posición incorrecta
Verificar que NFC esté activado en Ajustes. Cambiar la posición del teléfono sobre el tag — buscar la zona NFC (varía por modelo).

Error al escribir

Causa: Tag ya bloqueado o chip dañado
Si el tag fue bloqueado previamente, es imposible sobreescribirlo. Descartar y usar un tag nuevo. Reportar al inventario.

URL incorrecta escrita

Causa: Error de copy/paste en la URL
Si el tag no está bloqueado aún, puedes sobreescribirlo. Si ya fue bloqueado, descartar. Siempre verificar ANTES de bloquear.

La página abre con error 404

Causa: Código no existe en el sistema o sistema apagado
Verificar que el servidor esté corriendo. Confirmar que el lote fue activado en el panel admin. La URL debe tener el formato exacto.

El tag no se lee desde el exterior de la pieza

Causa: Material metálico entre el tag y el lector, o tag mal pegado
Reubicar el tag evitando herrajes metálicos, broches o placas. La zona interior discreta textil es la mejor zona.

iPhone no lee el tag

Causa: iPhone 6 o anterior (sin NFC nativo para URL)
iPhone 7 o superior necesario. En iPhone 10 o superior el NFC es automático. En iPhone 7-9 puede requerir abrir la cámara y tocar la notificación.
