Everything you need to program, verify and place NFC chips in QUIE® pieces. No prior technical knowledge required.

NFC · Chip: NTAG213 · 144 bytes · 13.56 MHz · Compatible iOS + Android

Step 0 — Before you start

What app to use?

NXP

NFC TagWriter by NXP RECOMMENDED

Android · Free · NXP Semiconductors

The official app from the chip manufacturer. Fast, reliable and free. Allows writing, reading and locking tags. This is the one we use in QUIE production.

NFC

NFC Tools Pro

Android + iOS · ~$4 USD

The most complete app on the market. Allows writing, reading, locking and viewing all chip data. Ideal if you also work from iPhone.

iOS

NFC.cool Tools

iOS iPhone 7+ (iOS 13+) · Free

Free option for iPhone. Can read and write URIs on NTAG213. Fewer features than NFC Tools Pro but works for basic production.

📌 Important: NFC must be ENABLED on your phone.
Android: Settings → Connections → NFC → Enable
iPhone: Works automatically on iPhone 7 or higher with iOS 11+

Full process

Programming a Tag — Step by Step

1

Prepare the batch CSV

Download the CSV from the admin panel. Each row is a code with its URL.

  • Go to Admin Panel → Batches → CSV
  • Open the file in Excel or Google Sheets
  • Sort the physical tags in the same order as the CSV
  • Work left to right, row by row
💡 The CSV contains: codigo_nfc, url_landing, modelo, color. The column you will copy is url_landing.
2

Open NFC TagWriter by NXP

Create a new URI dataset with the URL of the first code.

  • Open the app → "Write tags"
  • Tap "New dataset"
  • Select "URI"
  • Paste the URL from the CSV:
https://quie-nfc-platform.vercel.app/v/QUIE-A7B3X9-42
  • Tap "Save & Write"
💡 Copy the full URL including the https://
3

Bring the tag to the phone

Keep the chip flat and still under the phone until you hear confirmation.

  • Place the NFC tag on a flat surface
  • Bring the center or upper back of the phone over it
  • Hold still for 1–2 seconds
  • The app will show: "Tag written successfully"
Warning: do not move the phone while writing. If it fails, try again. After 3 failures, the tag may be damaged.
4

Verify the write

Read the written tag and confirm the URL is correct before locking.

  • Close the write screen
  • Bring the phone close to the same tag
  • A browser notification should appear
  • Confirm the URL matches the CSV
💡 You can also open NFC Tools, read the tag and see the URL in the "Records" tab.
5

Lock the tag (anti-counterfeit)

Once verified, lock it so nobody can overwrite it.

  • In NFC TagWriter: "Other functions" → "Lock tag" → Confirm
  • In NFC Tools Pro: Read tag → "More" → "Make read-only"
Warning: locking is IRREVERSIBLE. A tag locked with the wrong URL must be discarded. Always verify BEFORE locking.
6

Attach the tag to the piece

Interior position, discreet inner zone, flat and free from metal interference.

  • Inside the piece, discreet front inner zone
  • Avoid metal stitching — it interferes with the signal
  • If the tag has no adhesive: use industrial double-sided tape
  • The tag must be completely flat
  • Do a final scan from the outside to confirm readability
💡 Alternative: under the size/brand label on the interior.
7

Mark as done and continue

Move to the next code in the CSV and repeat from step 2.

  • Mark the CSV row as OK completed
  • Move to the next code in the CSV
  • Repeat from step 2
  • Each tag takes ~15 seconds
💡 Estimated time: 50 tags = 12–15 min · 100 tags = 25–30 min

Troubleshooting

Common Errors

Tag not detected

Cause: NFC disabled on the phone or incorrect position
Check that NFC is enabled in Settings. Try different positions of the phone over the tag — the NFC zone varies by model.

Write error

Cause: Tag already locked or chip damaged
If the tag was previously locked, it cannot be overwritten. Discard and use a new tag. Report to inventory.

Wrong URL written

Cause: Copy/paste error in the URL
If the tag is not yet locked, you can overwrite it. If already locked, discard it. Always verify BEFORE locking.

Page opens with 404 error

Cause: Code does not exist in the system or server is down
Verify the server is running. Confirm the batch was activated in the admin panel. The URL must follow the exact format.

Tag not readable from outside the piece

Cause: Metal material between tag and reader, or tag attached incorrectly
Reposition the tag away from metal hardware, snaps or plates. The textile inner zone is the best location.

iPhone does not read the tag

Cause: iPhone 6 or older (no native NFC for URLs)
iPhone 7 or higher is required. On iPhone 10 or higher NFC is automatic. On iPhone 7–9 you may need to open the camera and tap the notification.
